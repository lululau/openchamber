# OpenChamber 项目概览

## 项目定位

**OpenChamber** 是 [OpenCode](https://opencode.ai) 的富交互式用户界面，提供桌面端、浏览器端和移动端的全平台支持。

### 目标用户

- **AI 辅助编程开发者** - 使用 AI 代理进行代码编写、审查和重构的开发者
- **远程团队协作者** - 需要跨设备访问 AI 编程环境的团队成员
- **OpenCode 高级用户** - 需要比 CLI 更强大、更可视化界面的 OpenCode 用户
- **多项目管理员** - 同时管理多个项目和多个 AI 代理会话的技术负责人

## 核心功能特性

### 1. 多平台运行时

- **桌面应用**（macOS）：原生体验，支持 SSH 远程连接、项目动作、多窗口工作流
- **Web 应用**：支持 PWA 安装，可通过 Cloudflare Tunnel 实现远程访问
- **VS Code 扩展**：编辑器内集成，直接在编码环境管理 AI 会话

### 2. 可分支的聊天时间线

- `/undo` 和 `/redo` 命令支持撤销/重做任何对话轮次
- 从任意轮次一键创建分支，探索不同方案而不丢失进度
- 并行多代理运行，使用独立工作树进行安全对比

### 3. GitHub 原生工作流

- 直接从 Issue 和 PR 启动会话，自动附带上下文
- 完整的 Git 侧边栏：暂存、提交、推送/拉取、分支管理
- PR 创建（含 AI 生成的描述）、状态检查、合并操作
- 多远程推送、支持 Fork 的 PR 创建
- Git 工作树集成：每个分支独立会话，带冲突处理的合并流程

### 4. 强大的代码审查能力

- 精美的差异查看器，支持堆叠/内联模式
- 大变更集的延迟加载优化
- 点击消息中的文件路径可直接跳转到精确行号
- 内联评论草稿功能，将反馈发送回 AI 代理

### 5. 远程访问与协作

- Cloudflare Tunnel 支持（Quick 和 Named 模式）
- QR 码快速入网，扫描即可连接
- 安全的一次性连接链接
- 桌面应用支持通过 SSH 连接远程 OpenChamber 实例

### 6. 项目动作管理

- 一键运行开发服务器
- SSH 端口转发配置
- 在本地打开远程 URL
- 可自定义项目命令，提升开发效率

### 7. 高级交互功能

- **语音模式**：语音输入和朗读回复，解放双手
- **计划/构建模式**：专用计划视图，起草和迭代步骤
- **Shell 模式**：通过前缀 `!` 执行命令，内联输出
- **Mermaid 图表**：内联渲染，支持复制/下载
- **消息分享**：将对话导出为图片

### 8. 个性化定制

- 18+ 内置主题，支持浅色/深色变体
- 自定义主题 JSON 文件，热重载无需重启
- 可配置的键盘快捷键
- 可调节字体大小、间距、圆角和布局
- 自定义项目图标（支持上传和自动发现）

## 解决的问题

### 1. **CLI 界面局限性**
OpenCode 原生仅提供命令行界面，OpenChamber 提供了更直观、更强大的图形化交互方式，降低了使用门槛。

### 2. **跨设备访问困难**
开发者经常需要在不同的设备上访问 AI 编程环境（如从沙发上的手机、路上的平板、办公室的电脑）。通过 Cloudflare Tunnel 和 PWA 支持，OpenChamber 让随时随地编码成为可能。

### 3. **探索多种方案的成本高**
在传统 AI 聊天中，尝试不同方案通常需要重新开始对话。可分支的时间线让开发者可以轻松探索多种实现路径。

### 4. **GitHub 工作流割裂**
开发者需要在 AI 工具和 GitHub 之间来回切换。OpenChamber 将 PR 创建、状态检查、合并等操作直接集成到界面中。

### 5. **多项目管理复杂**
同时处理多个项目时，会话管理和上下文切换变得困难。OpenChamber 提供了项目级别的会话组织和快速切换能力。

## 技术栈

- **前端**：React、TypeScript、Vite、Tailwind v4
- **状态管理**：Zustand
- **UI 组件**：Radix UI、HeroUI、Remixicon
- **服务端**：Express（Node.js/Bun）
- **桌面端**：Tauri v2
- **VS Code 扩展**：Extension Host + Webview

## 项目结构

```
packages/
  ui/        共享 React 组件、hooks、stores 和主题系统
  web/       Web 服务器（Express）+ 前端（Vite）+ CLI
  desktop/   Tauri macOS 应用（Web UI 的轻量级外壳）
  vscode/    VS Code 扩展（扩展宿主 + webview）
```
